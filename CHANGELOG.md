# 更新日志

## v1.1.3

- 新增 10 张压缩沉浸风景壁纸，供“沉浸风景玻璃”主题和“休鼾一下”模式随机使用。
- 设置弹窗新增卡片“透明度调整”和“高斯模糊值调整”，支持滑块自定义整体功能卡片玻璃质感。
- 今日天气卡片新增天气对应风景背景，并将常见城市名称优先显示为中文。
- 将休鼾、今天、新事项三个快捷操作移动到左侧日历下方，避免右侧毛玻璃层影响点击。
- 毛玻璃背景层改为点击穿透，并使用 macOS 原生视觉效果控制卡片模糊程度。

## v1.1.2

- 应用正式更名为“灵栖胶囊Capsule”，统一菜单栏、窗口、通知、文档和安装包名称。
- 默认皮肤主题改为“沉浸风景玻璃”，首次打开默认展示沉浸风格。
- 更新 Bundle ID 为新品牌命名，系统层面识别为灵栖胶囊Capsule。

## v1.1.1

- 优化今日灵感胶囊关键词提取，避免粗暴截断导致语义缺失。
- 新增更贴近业务语义的短关键词提炼能力，例如用户生态、平台对接、UGC社区、审核机制等。
- 新增“沉浸风景玻璃”主题，使用风景背景、暗色遮罩、透明玻璃卡片和高可读文字。
- 沉浸主题内置 3 张压缩风景背景图并随机切换，控制安装包体积。
- 优化文本输入链路，记事保存改为防抖和后台写盘，降低批量输入卡顿。
- 针对 Intel 架构降低背景动效负载，Apple Silicon 保留更完整视觉动效。
- 优化事项通知调度，单个事项变更时不再全量重排所有通知。

## v1.1.0

- 优化左侧日历卡片高度，按实际月份周数显示，减少留白。
- 新增 macOS 顶部状态栏快捷面板，支持速记今日事项、查看今日待办并唤起主页面。
- 优化多主题视觉系统，主题会影响背景插画、情绪文案、图标氛围和组件质感。
- 新增“休鼾一下”模式，支持 5 分钟全屏沉浸放松、随机柔和壁纸和主动退出。
- 新增每天首次打开的情感化开屏提醒，5 秒自动关闭，也可主动关闭。
- 顶部状态栏改为今日灵感胶囊快捷输入，内容同步到当天胶囊，并新增休鼾模式入口。
- 今日灵感胶囊新增卡通人物和灵感蓄力百分比反馈。
- 默认启动图标替换为新版透明图标，并生成 macOS 多尺寸图标资源。
- 系统通知补充通知图标附件，减少通知栏图标为空的情况。
- 新增 iOS27 毛玻璃质感主题。
- 新增天气之子、你的名字、鬼灭之刃、夏目友人帐四款动漫氛围主题，补充对应背景动效和主题图标语言。
- 今日灵感胶囊字数进度调整为 300 字上限。
- 今日灵感胶囊新增 4 个自动关键词胶囊和今日心情判断。

## v1.0.0

- 首次开源版本。
- 支持日历事项添加与按日查看。
- 支持 macOS 系统通知提醒。
- 支持多种提醒频率。
- 支持今日灵感胶囊与 Word/PDF 导出。
- 支持多主题一键换肤。
- 支持古风默认主题。
- 支持今日天气组件。
- 支持用户自定义运行时 Dock 图标。
- 支持 Intel 与 Apple Silicon 通用构建。
