# 隐私说明

灵栖胶囊 Capsule 是本地优先的 macOS 桌面小组件。

## 本地保存的数据

以下内容默认保存在用户本机的 Application Support 目录中：

- 提醒事项、提醒时间、频率和完成状态。
- 今日灵感胶囊文本。
- 历史胶囊汇总所需的本地记录。
- 主题、玻璃透明度、高斯模糊等界面偏好。
- 用户自定义运行时 Dock 图标。

应用不会将这些内容上传到开发者服务器。

## 网络请求

天气组件会请求第三方服务来获取当前城市和天气数据：

- `ipapi.co` 用于根据网络环境推断城市。
- `open-meteo.com` 用于获取天气信息。

如果网络不可用，天气卡片会显示重试状态，不影响提醒、灵感胶囊、历史胶囊和导出功能。

## 系统通知

应用会使用 macOS 本地通知提醒事项。通知标题、时间和必要的事项信息会交给系统通知中心展示。

## 文件导出

用户主动导出 Word 或 PDF 时，应用会在用户选择的位置生成文件。导出文件只保存在本机。

## 删除数据

当前版本不提供一键清空全部数据入口。可在应用内删除事项、清空某日灵感文本，或手动删除 Application Support 中的本地数据文件。
